ngx-codemirror Cursor bug demo

Working Senario: type {{"${"}} and hit ctrl+space menu will open. Ex: "menu will open here {{"${"}} 'ctrl+space'"

Failure Senario: type "(" before {{"${"}} and hit ctrl+space menu will not open. Ex: "menu will not open here( {{"${"}}'ctrl+space'"

Note: if "(" is at start of line then it works as expected, But when it is added in between of any literal it fails.
